# API Reference

# Main Features
- Movie List 
- Filter with genre 
- Pagination with paging3
- Movie detail
- Artist info and artist detail
- Bottom navigation
- Search filter for movie
- Genre movie in the navigation drawer


## Architecture
  - MVVM Architecture (Model - ComposableView - ViewModel)
  - Repository pattern
  - Hilt - dependency injection
